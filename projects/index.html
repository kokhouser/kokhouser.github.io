Android Simplified Drawer (2014)


An Android library which provides a simple activity class to extend from to enable both left and right navigation drawers.

I worked on this library using intermediate knowledge of the Android SDK garnered from a three-month internship developing Android applications.

The current version of ASD supports extremely basic drawer functionality. It removes a lot of boilerplate code needed to make navigation drawers work.

A DrawerActivity class is provided to be extended by any activity that uses a drawer.

A DrawerManager class is also provided to be used to initialize drawers.

As of this version, ASD provides basic, one-layout type of drawer items, containing a single icon and its label. In the future, I hope to provide support for custom drawer item types on both drawers (left and right).

StringManip (2014)


A helper string class for C++ that adds string functionality from Ruby and Objective-C Foundation library that aims to alleviate some problems with string manipulation that C++ inherently has.

This was developed as a project for an Object-Oriented Design class in Spring 2014.

As of writing, the functions available through the helper class include:

chomp(String x): That returns string x with trailing endlines removed.
chomp(String x, String substring): Filters and removes substring from string x.
toSentence(String* stringArray, int n): Accepts a string array of size n and returns a sentence using the words in the array.
toSentence(String* stringArray, int n, String modifier): Same as above, with a custom sentence modifier.
toUpper(String x): Coverts all characters in the string x to uppercase.
capitalize(String x): Obtain profits. Capitalizes the first character in string x.
replaceOccurences(String x, String oldStr, String newStr): Given strings x, oldStr and newStr, replaces all occurences of oldStr with newStr.
explode(String x, String delimiter): Given string x, returns a vector containing all words in the string, seperated by given delimiter.
implode(Vector x, String joiner): Given a vector x, returns a string containing all words in the vector, seprated by given joiner.
wordCount(String x): Returns number of words in string x as an integer.
